Формулы рандомизации

Здесь объясняется, как рассчитываются вероятности при прямом выборе и при обратном выборе (по выбыванию) с учётом весов участников.

Обозначения

  • Для каждого участника i задан исходный вес w_i (если задано 0 или отрицательное, берётся минимум 1).
  • Эффективный вес: w*_i = max(1, w_i).

Прямой выбор (один победитель)

При прямом типе события выбирается победитель одним жребием со взвешенной вероятностью.

Вероятность выбора участника i:

P(выбран i) = w*_i / (Σ_j w*_j)

То есть чем больше вес участника, тем выше его шанс быть выбранным победителем.

Обратный выбор (по выбыванию)

При обратном типе события система по очереди исключает участников, пока не останется один — он и является победителем.

Чтобы участники с большими весами реже выбывали, веса инвертируются по формуле:

w'_i = max_w − w*_i + 1, где max_w = max_k(w*_k) среди активных

Затем на каждом шаге участник на выбывание выбирается со взвешенной вероятностью по инвертированным весам:

P(выбывает i на шаге) = w'_i / (Σ_{k в активных} w'_k)
  • Если у участника максимальный исходный вес (w*_i = max_w), то w'_i = 1 — минимальный шанс выбыть.
  • Чем меньше исходный вес, тем больше w'_i и тем выше шанс выбыть раньше.

После исключения участника пересчитываются активный список и, при необходимости, max_w и веса w'_i. Процесс повторяется до одного оставшегося участника.

Связь с реализацией

В коде приложения (app/app.py) используются следующие ключевые шаги:

  1. Подготовка пар (id, вес) для активных участников.
  2. Функция взвешенного выбора по равновероятному целому числу r в диапазоне [1, Σ max(1, w)]: 
    # Упрощённо
r ∈ {1, 2, ..., T}, где T = Σ_j max(1, w_j)
Накопленная сумма S ← 0
Для каждого участника (id, w):
    S ← S + max(1, w)
    если r ≤ S: выбрать этого участника
  3. Для обратного выбора перед вызовом взвешенного выбора формируются инвертированные веса по формуле выше, затем выбранный участник помечается как «выбыл».

Примеры

Пусть даны три участника с весами: A=1, B=2, C=5. Тогда:

  • Прямой выбор: Σ=1+2+5=8. P(A)=1/8, P(B)=2/8, P(C)=5/8.
  • Обратный выбор: max_w=5. Инвертированные веса: A' = 5-1+1=5, B' = 5-2+1=4, C' = 5-5+1=1. На первом шаге P(выбывает A)=5/(5+4+1)=5/10, P(выбывает B)=4/10, P(выбывает C)=1/10.

Таким образом, участник с наибольшим исходным весом реже выбывает и чаще остаётся до финала.
