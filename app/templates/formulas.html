Формулы рандомизации

Здесь объясняется, как рассчитываются вероятности при прямом выборе и при обратном выборе (по выбыванию) с учётом весов участников.

Обозначения

  • Для каждого участника i задан исходный вес w_i (если задано 0 или отрицательное, берётся минимум 1).
  • Эффективный вес: w*_i = max(1, w_i).

Прямой выбор (один победитель)

При прямом типе события выбирается победитель одним жребием со взвешенной вероятностью.

Вероятность выбора участника i:

P(выбран i) = w*_i / (Σ_j w*_j)

То есть чем больше вес участника, тем выше его шанс быть выбранным победителем.

Обратный выбор (по выбыванию)

При обратном типе события система по очереди исключает участников, пока не останется один — он и является победителем.

Начиная с версии, где обновлён расчёт инверсии в run_event(), вероятности выбывания считаются на основе «инверсных» весов с мягкой степенной защитой больших весов:

Пусть w*_i = max(1, w_i), а Σ = Σ_k w*_k по активным участникам.
Инверсный вес:   w'_i ∝ Σ / (w*_i)^α,  где α = 0.5 (по умолчанию)
Нормировка:      P(выбывает i на шаге) = w'_i / (Σ_{k в активных} w'_k)
  • Чем больше исходный вес w*_i, тем меньше значение w'_i, следовательно, тем ниже шанс выбыть.
  • Параметр α = 0.5 делает защиту «сублинейной»: рост исходного веса уменьшает риск выбывания, но не слишком резко.
  • Масштабный множитель (например, ×100 и округление до целого) не влияет на вероятности — он используется только для удобства целочисленного выбора.

После исключения участника пересчитываются активный список, сумма Σ и, соответственно, инверсные веса w'_i. Процесс повторяется до одного оставшегося участника.

Связь с реализацией

В коде приложения (app/app.py) используются следующие ключевые шаги:

  1. Подготовка пар (id, вес) для активных участников.
  2. Функция взвешенного выбора по равновероятному целому числу r в диапазоне [1, Σ max(1, w)]: 
    # Упрощённо
r ∈ {1, 2, ..., T}, где T = Σ_j max(1, w_j)
Накопленная сумма S ← 0
Для каждого участника (id, w):
    S ← S + max(1, w)
    если r ≤ S: выбрать этого участника
  3. Для обратного выбора перед вызовом взвешенного выбора формируются инвертированные веса по формуле выше, затем выбранный участник помечается как «выбыл».

Примеры

Пусть даны три участника с весами: A=1, B=2, C=5. Тогда:

  • Прямой выбор: Σ=1+2+5=8. P(A)=1/8, P(B)=2/8, P(C)=5/8.
  • Обратный выбор (α=0.5): Σ=8. Инверсные веса пропорциональны Σ/√w: A'≈8/1=8, B'≈8/√2≈5.66, C'≈8/√5≈3.58. На первом шаге вероятности выбывания пропорциональны этим значениям: P(выбывает A)≈8/(8+5.66+3.58)≈0.46, P(выбывает B)≈0.33, P(выбывает C)≈0.21.

Таким образом, участник с наибольшим исходным весом реже выбывает и чаще остаётся до финала. Масштабирование на целые числа (×100 и округление) не меняет этих вероятностей.
